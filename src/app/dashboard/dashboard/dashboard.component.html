About Epidemic Simulator

The Epidemic Simulator is an tool designed to model the spread and impact of infectious diseases within a defined population. This simulator allows users to input various parameters related to the disease and population to observe how the epidemic progresses over time. The simulation provides valuable insights into infection rates, recovery, and mortality, helping users understand potential outcomes and plan accordingly.

Key Features

  • Customizable Simulations: Tailor the simulation parameters to match specific scenarios.
  • Detailed Results: Obtain comprehensive data on the number of infected, healthy, recovered, and deceased individuals over the simulation period.
  • User-Friendly Interface: Easy-to-use controls and clear visualizations of the simulation results.

How It Works

Simulation Parameters

  • Name: A descriptive name for the simulation scenario.
  • Population Size: The total number of individuals in the population.
  • Initial Infected Count: The number of individuals initially infected at the start of the simulation.
  • Reproduction Rate: The average number of new infections caused by each infected individual.
  • Mortality Rate: The proportion of infected individuals who succumb to the disease. This value should be between 0 and 1, with a maximum of two decimal places.
  • Recovery Time: The time in days it takes for an infected person to recover and gain immunity.
  • Mortality Time: The time in days from infection to death for those who do not recover.
  • Simulation Duration: The total time in days for which the simulation runs.

Example Usage

Create a New Simulation

Navigate to the simulation creation page. Enter the required parameters based on your scenario. Start the simulation to see how the disease spreads over time.

View Simulation Results

Access the results page to see detailed statistics and graphs showing the progression of the epidemic. Analyze the data to understand the impact of different parameters on the spread of the disease.

Update an Existing Simulation

Modify the parameters of an existing simulation to test different scenarios. Save the updated simulation and review the new results.
